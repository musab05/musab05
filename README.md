## 👋 About Me

I'm a full-stack developer from India who builds complete, end-to-end products — from real-time classroom platforms to low-level systems programming. I work across the full stack with an emphasis on performance, clean architecture, and shipping things that actually work.

Currently building a **SQL database engine from scratch in Rust** and a **Unity survival/horror game**. I enjoy solving hard problems across multiple layers of the stack.

---

## 🚀 Featured Projects

### 🎓 Academiq — Full Education Ecosystem
> `React` `Node.js` `WebRTC` `MongoDB`

A complete educational platform supporting public courses, institute management, live video classrooms, team collaboration, and an integrated AI learning assistant. Built with real-time features using WebRTC for live sessions.

---

### 🤝 Threadly — Professional Networking Platform
> `React` `Node.js` `PostgreSQL` `Maps API`

LinkedIn-inspired platform with posts, job listings, recommendations, and a **map-based job discovery** feature. Full relational data model with PostgreSQL.

---

### 🔐 Secure Share — Encrypted P2P CLI Tool
> `Rust` `CLI`

Fast, minimal, serverless CLI for encrypted peer-to-peer file sharing. No servers, no middlemen — pure Rust.

---

### 🤖 Chat Bot — AI Learning Assistant
> `Python` `LLM` `React`

In-context AI assistant integrated into Academiq for intelligent, context-aware student help. Bridges LLM capabilities with live course content.

---

### ✍️ Blog Vista — Blogging Platform
> `React` `Node.js` `MongoDB`

Full-featured blogging platform with rich text and **video blog support**.
